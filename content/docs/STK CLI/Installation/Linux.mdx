Para instalar a **última versão do STK CLI**, você precisa fazer o **download do instalador** e executá-lo no seu terminal.

### **Distribuições Homologadas**
O StackSpot CLI foi homologado para funcionar nas seguintes distribuições do Linux:
- Ubuntu 20.04 
- Ubuntu 18.04

Não há requisitos de instalação para outras distribuições, mas a compatibilidade do StackSpot CLI pode não ser total.
No momento, a distribuição **`Amazon Linux`** não é compatível com o StackSpot CLI.

### **Requisitos**
Veja abaixo os requisitos para usar o STK CLI:
- <a href="https://git-scm.com/downloads" target="_blank"><strong><ins>Instalar o Git - a partir da versão 2.32.</ins></strong></a>  

### **Passo a passo de Instalação**    

##### **Passo 1.** Faça o download do instalador

> **Acesse a seguinte URL:** <ins>**www.stackspot.com/download**</ins>  


##### **Passo 2.** Pelo terminal navegue para pasta onde você baixou o pacote de instalação.

Para o arquivo **`stk.deb`**, execute o seguinte comando:
```
sudo dpkg --install stk.deb
```

Para o arquivo **`stk.rpm`**, execute o seguinte comando:
```
rpm -i stk.rpm
```

##### **Passo 3.** Reinicie o terminal após a instalação, isso garante que o comando stk seja reconhecido.

Caso o terminal esteja aberto, feche e abra novamente após instalação, isso garante que o comando stk seja reconhecido.

<alert color="warning">
É necessário finalizar o processo do terminal antes de reiniciá-lo.
</alert>

No terminal execute o comando:
```
stk –-version
```
Se tudo estiver configurado, o terminal irá retornar o número da versão atual do STK CLI.

##### **Passo 4.** Execute o comando abaixo para concluir a inicialização do STK CLI:

```text
stk init
```

### **Passo a passo de Desinstalação**   

##### **Passo 1.** Entre no terminal.

Caso você tenha feito a instalação usando o pacote **`.deb`**, execute o seguinte comando:
```
sudo dpkg -r stkcli 
```
Ou o comando:
```
sudo apt-get remove stkcli
```

Caso você tenha instalado usando o pacote **`.rpm`**, execute o seguinte comando:
```
rpm -e stkcli
```

Em ambos os casos, uma mensagem aparecerá para confirmar a desinstalação antes que ela se inicie.
