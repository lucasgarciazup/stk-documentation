# **O que há de novo?**  

A nova versão traz algumas melhorias como correções de erros e ajustes para a compatibilidade com plugins de IDEs.  

## **Mudanças**

- Ajustes de funcionalidades para compatibilidade com plugins de IDE.
- Exportação do módulo Stack-Engine como biblioteca.
- Alteração do nome da pasta onde as stacks são importadas. A pasta **`.stk/plugins`** foi renomeada para **`.stk/stacks`**.

## **Resolução de problemas (Bug Fixes)**

- Correção do erro de codificação (encoding) na criação de apps templates.
- Correção de erro ao executar o comando [**<ins>`stk list template`</ins>**]({{< ref path="/docs/References/Commands List/stk-list-template.md" >}}), onde os arquivos sem a codificação adequada não eram exibidos corretamente. Agora uma mensagem de erro é apresentada e informa que a codificação dos arquivos deve ser `UTF-8`, além do nome do arquivo que não possui a codificação adequada.
- Remoção da mensagem de sucesso ao aplicar um plugin quando o comando era interrompido ou mal sucedido. 
- Ajustes nas configurações dos arquivos shell usados na [**<ins>instalação para MacOS</ins>**]({{< ref path="/docs/STK CLI/Installation/Macos.md" >}}).
