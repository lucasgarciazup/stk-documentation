# **O que há de novo?**
A nova versão traz novas atualizações como instalação online e autoproxy. 

## **Features**
- **Novo comando:** 
    - [**`stk upgrade`**]({{< ref path="/docs/References/Commands List/stk-upgrade.mdx" >}}).

## **Mudanças**
- Melhorias de performance com a remoção do PyInstaller.

## **Resolução de problemas (Bug Fixes)**
- Ajustes no **`--help`** dos comandos de delete: 
    - [**`stk delete plugin`**]({{< ref path="/docs/References/Commands List/stk-delete-plugin.mdx" >}})
    - [**`stk delete template`**]({{< ref path="/docs/References/Commands List/stk-delete-template.mdx" >}}) 
    - [**`stk delete stack`**]({{< ref path="/docs/References/Commands List/stk-delete-stack.mdx" >}}).
- Ajustes no texto da flag **`--stackfile`** nos comandos:
    - [**`stk create app`**]({{< ref path="/docs/References/Commands List/stk-create-app.mdx" >}}) 
    - [**`stk create env`**]({{< ref path="/docs/References/Commands List/stk-create-env.mdx" >}}).
- Correção de erro no processamento do **`--stackfile`**.
- Correção nos comandos de **`stk delete`** para criação de commits de repositório separados e nos testes integrados. 
- Correção do charset/encoding do `stkcli` no Windows (cmd e Git Bash).
