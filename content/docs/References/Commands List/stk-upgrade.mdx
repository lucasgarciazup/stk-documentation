O comando **`stk upgrade`** atualiza a versão do CLI quando há uma nova disponível.

### **Como funciona?**
Quando o comando **`stk upgrade`** é rodado no terminal, ele atualiza o CLI para a versão mais recente. Se houver uma nova versão, a seguinte mensagem será exibida:

```
Your STK CLI version 3.1.0 is outdated. The new version 3.1.1 is available.
To upgrade run the command: 'stk upgrade'.
```


### **Parâmetros**
Este comando não possui parâmetros.


### **Exemplo**
Veja o exemplo abaixo:

1. Atualização do CLI para a última versão disponível:
```
stk upgrade
```
