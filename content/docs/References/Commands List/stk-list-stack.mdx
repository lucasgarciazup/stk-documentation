O comando **`stk list stack`** lista as stacks disponíveis vindas da instalação ou que foram adicionadas pelo comando **`stk add stack`**. O comando também é capaz de filtrar as stacks pelo nome delas ou pela descrição através do parâmetro **`--filter`**.

### **Como funciona?**
O comando exibe uma lista de stacks disponíveis.

Esta lista contém:
- Nome da stack.
- Descrição da stack.

### **Parâmetros**
Confira a opção disponível no comando:


| Parâmetro | Descrição |
| ----------- | ----------- |
| `--filter` ou `-F` | Filtra as stacks por valores que estão no nome delas ou na descrição. |
