O comando **`stk login`** realiza o login numa conta do StackSpot Enterprise. Realizar o login é uma premissa para execução de alguns comandos, como: **`stk list action`**.

### **Como funciona?**
O comando necessita que seja informado seu E-mail da organização. Ao informar seu E-Mail o comando irá abrir seu navegador e pedir pelo preenchimento de seus dados para gerar uma sessão.

### **Parâmetros**
Confira a opção disponível no comando:

| Parâmetro | Descrição |
| ----------- | ----------- |
| `EMAIL` | E-Mail utilizado para fazer o login na conta Enterprise.   |

### **Exemplo**
Você pode usar esse comando de qualquer local na sua máquina para realizar o login na conta Enterprise.
