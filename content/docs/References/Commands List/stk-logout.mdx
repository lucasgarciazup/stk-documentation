O comando **`stk logout`** realiza o logout da conta do StackSpot Enterprise. Realizar o logout inativa a sessão criada através do comando **`stk login`**. 
Realizar o logout impedirá o uso de alguns comandos como: **`stk list action`**.

### **Como funciona?**
O comando irá revogar os acessos do usuário logado.

### **Exemplo**
Você pode usar esse comando de qualquer local na sua máquina para realizar o logout da conta Enterprise.
