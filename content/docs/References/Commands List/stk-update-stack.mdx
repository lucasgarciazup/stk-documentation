O comando **`stk update stack`** atualiza todas as stacks disponíveis na StackSpot ou qualquer outra na sua aplicação. Esta atualização é necessária quando o nome da stack a ser atualizado é omitido.

### **Como funciona?**
É só rodar o comando **`stk update stack`** no terminal para atualizar as stacks de plugin localmente.
Você pode usar o argumento **<`nome_stack`>**  após o comando para atualizar uma stack específica.


### **Parâmetros**
Confira a opção disponível no comando:

| Parâmetro | Descrição |
| ----------- | ----------- |
| `STACK_NAME` | Define o nome da stack que você pode criar. |


### **Exemplo**
Veja o exemplo abaixo:

1. Atualiza todas as stacks:
```
stk update stack
```
