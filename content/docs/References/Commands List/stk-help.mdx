O STK CLI possui a opção **`--help`** para mostrar mais informações sobre os comandos e seus parâmetros.

### **Como funciona?**
Escreva no seu terminal o comando que precisa saber mais informações e logo depois adicione **`--help`**. Veja os exemplos abaixo:


### **Exemplos**
Veja exemplos abaixo:

- **`stk init -–help`**
- **`stk create catalog -–help`**
- **`stk create plugin -–help`**
