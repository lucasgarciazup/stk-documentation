O STK CLI possui a opção **`--version`** para mostrar a versão atual instalada.

### **Como funciona?**
O comando exibe no terminal a versão atual que está instalada.


### **Parâmetros**
Confira a opção disponível no comando:


| Parâmetro | Descrição |
| ----------- | ----------- |
| `--version` ou `-v` | Apresenta no terminal a versão atual do STK CLI instalada em sua máquina. |


### **Exemplo**
Veja o exemplo abaixo:

1. Confira a versão atual do STK CLI instalada:
```
stk --version
```
