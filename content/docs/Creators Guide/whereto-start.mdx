## **Qual a diferença de Monorepo e Multirepo?**

A criação de uma Stack pode ser feita usando uma dessas duas estratégias, ambas oferecem vantagens e desvantagens no seu uso.

- **Monorepo**: Neste método a Stack e todos os seus elementos são criados no mesmo repositório, tornando-os exclusivos dessa Stack. Trata-se de um método mais simplificado e rápido para criar uma Stack completa (contendo Templates, Plugins e Stackfiles). Para saber como utilizar este método, leia o guia [<ins>**Criar uma Stack**</ins>]({{< ref path="/docs/Creators Guide/howto-create-stack.mdx" >}}).

- **Multirepo**: Nesta abordagem cada elemento é criado em um repositório separado para cada um. Você pode criar Plugins e Templates separados, sem criar uma Stack antes. Neste caso os Plugins e Templates são versionados de forma isolada, e deixam de ser exclusivos da Stack, permitindo serem compartilhados entre outras Stacks.  

Para mais informações de como utilizar esse método, leia os guias abaixo:

- [<ins>**Criar Templates**</ins>]({{< ref path="/docs/Creators Guide/howto-create-template.mdx" >}})
- [<ins>**Criar Plugins**</ins>]({{< ref path="/docs/Creators Guide/howto-create-plugin.mdx" >}}).
